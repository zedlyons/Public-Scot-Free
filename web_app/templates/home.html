PROJECT SCOT-FREE

Hello. This web-page is the culmination of PROJECT SCOT-FREE, an attempt to statistically predict which murders remain unsolved and which killers are brought to justice. The model accessed throughout this website is a random forest trained on FBI NIBRS data (BSS = 0.275 at final evaluation). As it has been trained exclusively on data from the six New England states from the years 2022-2024 (inclusive), its predictive power beyond these geographic and temporal bounds is limited. The source code can be found at the Github repository linked at the bottom of the page.
